# scanpar
C-like scanner and parser that takes in C file and analyses the file itself. 

### [Updated] Create Executable
```
gcc scan.c parser.c
```
### [Old] Create Executable
``` 
make 
```
### [Update] Run
#### Scanner Only 
```
./a.out test.c
```

#### Parser & Scanner 
```
Note: 'y' signifies yes for parser
./a.out test.c y
```

### [Old] Run
```
./scan [insert file name with extensions] 
./scan test.c
```
